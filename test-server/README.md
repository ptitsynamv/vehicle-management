# Simple Test Server

This is a simple Node.js test server for managing vehicles.

## Prerequisites

- Node.js v22.15.1
- NPM v10.9.2

## Setup

1. Install dependencies:

```bash
npm install
```

2. Start the server:

```bash
npm start
```

## API Endpoints

- **GET /vehicles** → List all vehicles
- **GET /vehicles/\:id** → Get a vehicle by ID
- **POST /vehicles** → Add a new vehicle
